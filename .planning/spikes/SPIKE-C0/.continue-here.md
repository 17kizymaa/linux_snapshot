## What was built (C0 + C1)

1. **FTS5 virtual table** (`decisions_fts`) with trigram tokenizer, sync triggers + backfill
2. **Taxonomy columns** on `decisions`: kind, scope, confidence, tags, expires_at, pinned
3. **Ranked recall** replacing LIKE: BM25 + recency decay + project boost + pinned boost + error boost + source boost
4. **Explainable scores**: each result has `_score_breakdown` per component
5. **Kind auto-detection**: `remember()` maps category → kind via `category_to_kind()` with aliases (prefer→preference, etc.)\
6. **FTS5 query escaping**: `_escape_fts5_query()` splits on whitespace, double-quotes each term — prevents operator injection from dashes, quotes, etc.
7. **Deduplication**: `_dedup_memories()` uses character-bigram Jaccard similarity (threshold 0.75) after ranking, before final limit
8. **SessionStart kind grouping**:session_start.py groups by pinned/preference/decision/error/procedure/note instead of just category
9. **Migration**: `migrate_taxonomy()` and `migrate_fts5()` — safe, idempotent, called from `AwarenessStore._migrate()`
